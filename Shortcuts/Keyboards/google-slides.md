### Common actions

```
Cmd + /         - show common keyboard shortcuts
Ctrl + m        - new slide
Cmd + d         - duplicate
Cmd + z         - undo
Cmd + y         - redo
Cmd + Sft + z   - redo
Cmd +  c        - copy
Cmd + x         - cut
Cmd + v         - paste
Delete          - delete
Cmd + f         - find

```

### Navigation

```
Cmd + enter      - present slides
Enter            - play the selected video
Cmd + opt + +    - zoom in
Cmd + opt + -    - zoom out
Esc              - exit the current mode
```

### Text

```
Cmd + b         - Bold
Cmd + i         - italic
Cmd + u         - underline
Cmd + ,         - subscript
Cmd + .         - superscript
Cmd + sft + x   - strikethrough
Cmd + sft + >   - increase font size
Cmd + sft + <   - decrease font size
Cmd + sft + l   - left align
Cmd + sft + r   - right align
Cmd + sft + c   - center align
Cmd + sft + j   - justify
```
